## Requirements

* Yarn 0.21.3+

## Setup

* Run `yarn install` or `npm install` to install dependencies.


## Run Steps

* Note: Currently for dev mode the UI and controller must be run seperately.
* You can run `yarn dev` or `npm run dev` to start the controller.
